Function Details

delete/2

delete(Key::atom(), Db::[tuple()] | []) -> [tuple()] | []

Delete record with the key specified from database. If there is record with the key specified in the database, it is removed and new database is returned. Otherwise old database is returned.

destroy/1

destroy(Db::[tuple()]) -> ok

Destroy database. Returns 'ok'.

match/2

match(Element::atom(), Db::[tuple()] | []) -> [tuple()] | []

Find record in database by element value. Returnes [] if record with the element specified was not found. Otherwise returns list of keys correspondent to the element specified.

new/0

new() -> []

Create new database. Returns [] - an empty database

read/2

read(Key::atom(), Db::[tuple()] | []) -> {ok, atom()} | {error, instance}

Find record in database by key. Returns {ok, element} tuple - if record with key specified is found in the database. Otherwise, {error, instanse} tuple is returned.

write/3

write(Key::atom(), Element::atom(), Db::[tuple()] | []) -> [tuple()]

Add new record to database. If element with the specified key is already present in database, nothing will be added and old database will be returned. Otherwise, new database is returned
